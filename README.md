## Overview

A comprehensive collection of genetic algorithm research iterations exploring optimization approaches for the discrete logarithm problem on secp256k1. Each variant tests different fitness functions, crossover operators, and selection strategies.

## Approach

Genetic algorithms applied to ECDLP explore whether evolutionary optimization can find structure in the key space that pure mathematical approaches miss. While GA cannot solve ECDLP efficiently (the problem is believed to be hard), the research reveals interesting properties of the fitness landscape.
